Pattern 3

For existing installations (e.g. Nikola Tech) where use of 3rd party cloud is required by vendor, data flows from sensor data via cellular JSON, thereby bypassing existing Lilly Manufacturing network checks. Any security offerings that may exist on the vendor cloud will have to be accepted.

After analysis at the vendor cloud, the results are pushed to public cloud platform using cloud-2-cloud connectivity. If the vendor so supports, raw data also is collected in the public cloud and used for building and training predictive models

Selective data (e.g. raw time and freq. domain data) are pushed to the Enterprise Data lake

Quick setup for trying out ideas using the end-2-end vendor offering

Not suited for Red or Amber category of data

Ownership and maintainability of solution components (software, interface, network etc.) may not be straightforward.
