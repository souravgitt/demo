Pattern 4

Data from smart devices or sensors (high speed or legacy) data will be collected at the edge gateway via HART (wireless/ wired)

Based on the rules applied at the edge interface, selective data (e.g. processed vibration analysis) or all data (e.g. raw time and frequency domain data) will flow to the Enterprise data Lake via the Manufacturing firewall using HTTPS.

This data along with other contextual data from IBM PDA is pushed to the public cloud platform using rest API's, for building and training analytical models. The data security offerings by the cloud platform will be used to route this data.

Clear demarcation of ownership

Ease of device manageability and new device provisioning

Future ready capability of intelligence at edge

Cloud connectivity that the device or sensor may offer, is not being leveraged
