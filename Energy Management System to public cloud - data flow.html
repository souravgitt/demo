Pattern 2

Selective utilities data (e.g. energy values) from local OSi-Soft gets collected at the Enterprise OSi-Soft via the firewall, using the Pi-File connector from OSi-Soft.

 

This data along with other contextual data from IBM PDA is pushed to the public cloud platform using rest API's, for building and training analytical models. The data security offerings by the cloud platform will be used to route this data.

Utilization of existing data flow, hence no additional network tweaking

Available rest-API's in OSi-Soft to push data directly to cloud, not being leveraged
